## Omitting `std::`

If you have trouble with the `std::` prefix, consider using it
locally, **always inside a function** (as shown in the example):

```cpp title="ðŸ”¹ Acceptable use"
#include <iostream>
int main()
{
	// highlight-next-line
	using namespace std;
	cout << "Hello, World!";
}
```

I emphasize this because it is possible to place `using namespace std;` at the top of your file,
but **don't do this** (even though it's all over the Internet!), as it can lead to mysterious errors stemming from ambiguity.
This is because `std` has a **huge** amount of names that you will commonly use in your code.

:::tip Good practice
It's best not to omit the `std::` prefix.
:::

### Know the difference!

There are two ways to omit the `std::` prefix:

```cpp
using namespace std;
```

and

```cpp
using std::cout;
```

`using namespace std` and `using std::cout`

- `using namespace std;` means that you don't need to prepend `std::` for any member of the standard library you use.
Common items you may see on your journey and in later tutorials are `cout`, `cin`, `string`, `array`, and `vector`.

- `using std::cout;` means that only `cout` has the privilege to not require `std::` in front of it. You can do this for multiple items at once like:
`using std::cout, std::cin, std::string;`


Generally, it is recommended to use the `using std::cout;` style over `using namespace std;` if you wish to omit `std::`.

An error is caused if you omit `std::` without the appropriate `using` statement.

Forgetting to add the second quotation mark leads to C++ thinking the rest of your program is text.

## Using `"` inside of text

You may be tempted to put quotes inside a message when outputting it to the console.
However, remember that C++ considers everything inside of `"..."` quotes to be text.

Thus, if you write:
```cpp
"Hi! My name is "Slim Shady". Nice to meet you!"
```

C++ will interpret this as:
```plaintext
"TEXT" C++ CODE "TEXT"
```

You can prevent this by using the `\` backslash character to *escape* the quote.

```cpp
"Hi! My name is \"Slim Shady\". Nice to meet you!"
```
