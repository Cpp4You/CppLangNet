# Aliases

```cpp title="Creating own alias"
using CustomType = int;
```
```cpp title="Example usage"
CustomType a = 1;
```

## Useful aliases

| Type | Equivalent (usually) | Description |
| ---- | ----- | ----- |
| `int8_t` | `char` | Integer 8-bit |
| `int16_t` | `short` | Integer 16-bit |
| `int32_t` | `int` | Integer 32-bit |
| `int64_t` | `long long` | Integer 64-bit |
| `uint8_t` | `unsigned char` | Integer (unsigned) 8-bit |
| `uint16_t` | `unsigned short` | Integer (unsigned) 16-bit |
| `uint32_t` | `unsigned int` | Integer (unsigned) 32-bit |
| `uint64_t` | `unsigned long long` | Integer (unsigned) 64-bit |
