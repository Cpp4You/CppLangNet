# std::generate() algorithm

<SwitchView content={{
  'since-cpp20': <Signature_SinceCpp20 />,
  'since-cpp17': <Signature_SinceCpp17 />,
  'until-cpp17': <Signature_UntilCpp17 />
}} />

- **(1)**
  Assigns each element in range [`first`; `last`) a value generated by the given function object `g`.

 - **(2)** Same as **(1)**, but executed according to `policy`.
  :::important Overload Resolution
  These overloads participate in overload resolution only if
  <Until v='cpp20'> <code>std::is_execution_policy_v&lt;std::decay_t&lt;ExecutionPolicy&gt;&gt;</code> </Until> <Since v='cpp20'> <code>std::is_execution_policy_v&lt;std::remove_cvref_t&lt;ExecutionPolicy&gt;&gt;</code> </Since> is <code>true</code>.
  :::

```cpp

```

### Return value

**(none)**

### Complexity

Exactly `std::distance(first, last)` invocations of `g()` and assignments.

### Exceptions

The overloads with a template parameter named `ExecutionPolicy` report errors as follows:
- If execution of a function invoked as part of the algorithm throws an exception and `ExecutionPolicy` is one of the [**standard policies**](#), [`std::terminate`](#) is called.
	For none other `ExecutionPolicy`, the behavior is <Tooltip title={<Term_ImplementationDefined/>}>**implementation-defined**</Tooltip>.
- If the algorithm fails to allocate memory, `std::bad_alloc` is thrown.

### Possible implementation

<details>
<summary>generate (1)</summary>

```cpp
template<class ForwardIt, class Generator>
constexpr //< since C++20
void generate(ForwardIt first, ForwardIt last, Generator g)
{
    for (; first != last; ++first)
        *first = g();
}
```

</details>

### Examples

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <algorithm>
#include <iostream>
#include <vector>

void println(std::string_view fmt, auto const& v)
{
    for (std::cout << fmt; auto const& e : v)
        std::cout << e << ' ';
    std::cout << '\n';
};


int f()
{
    static int i;
    return ++i;
}

int main()
{
    std::vector<int> v(5);

    std::generate(v.begin(), v.end(), f);
    println("v: ", v);

    // Initialize with default values 0,1,2,3,4 from a lambda function
    // Equivalent to std::iota(v.begin(), v.end(), 0);
    std::generate(v.begin(), v.end(), [n = 0] () mutable { return n++; });
    println("v: ", v);
}
```
