# std::vector pop_back() method

<SwitchView content={{
    'since-cpp20': <Method_SinceCpp20 />,
    'until-cpp20': <Method_UntilCpp20 />,
}} />

Removes the last element of the container.

Calling `pop_back` on an empty container results in <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.

:::danger Invalidation
Iterators and references to the last element, as well as the `end()` iterator, are **invalidated**.
:::

### Parameters

(none)

### Return value

(none)

### Complexity

Constant.

### Exceptions

(none)

### Example

<div>

```cpp title='Main.cpp' showLineNumbers
#include <vector>
#include <iostream>
 
template<typename T>
void print(T const & xs)
{
    std::cout << "[ ";
    for(auto const & x : xs) {
        std::cout << x << ' ';
    }
    std::cout << "]\n";
}
 
int main()
{
    std::vector<int> numbers;
 
    print(numbers); 
 
    numbers.push_back(5);
    numbers.push_back(3);
    numbers.push_back(4);
 
    print(numbers); 
 
    numbers.pop_back();
 
    print(numbers); 
}
```
