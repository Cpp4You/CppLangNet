# std::array back() method

<VersionTabs content={{
	'since-cpp17': <Method_SinceCpp17 />,
	'since-cpp14': <Method_SinceCpp14 />,
	'until-cpp14': <Method_UntilCpp14 />,
}}/>

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the last element in the container.

<span className="inline-danger">Calling <code>back</code> on an empty container results in an <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.</span>

### Parameters

(none)

### Return value

Reference to the last element.

### Complexity

Constant.

### Notes

For a container `c`, the expression `c.back()` is equivalent to `*std::prev(c.end())`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <array>
#include <iostream>
 
int main()
{
    std::array<char, 6> letters {'a', 'b', 'c', 'd', 'e', 'f'};
 
    if (!letters.empty()) {
        std::cout << "The last character is '" << letters.back() << "'.\n";
    }  
}
```

</div>
<div>

```plaintext title="Possible output"
The last character is 'f'.
```
