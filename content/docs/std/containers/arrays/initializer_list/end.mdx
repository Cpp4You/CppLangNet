# std::initializer_list end() method

<SwitchView content={{
	'since-cpp14': <Method_SinceCpp14 />,
	'until-cpp14': <Method_UntilCpp14 />
}}/>

Returns a <Tooltip term={Term_ContAlias_Pointer}>pointer</Tooltip> to the last element in the initializer list.

If the initializer list is empty, the returned pointer will be equal to [`begin()`](begin).

### Parameters

**(none)**

### Return value

Pointer to the first character.

### Complexity

Constant - **O(1)**.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <initializer_list>

int main()
{
    static constexpr auto il = {42, 24};
    static_assert(*il.end() == 0x2A);
}
```
