If `std::allocator_traits&lt;allocator_type&gt;::propagate_on_container_swap::value` is `true`.
then the allocators are exchanged using an unqualified call to non-member swap.  
Otherwise, they are not swapped.

:::danger Undefined Behavior
If `get_allocator() != other.get_allocator()`, the behavior is <Tooltip title={Term_UndefinedBehaviour}>undefined</Tooltip>.
:::
