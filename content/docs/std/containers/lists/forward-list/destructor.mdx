# std::forward_list destructors

<SwitchView content={{
	'since-cpp11': <Method_SinceCpp11 />,
}} />

Destructs the forward_list. The destructors of the elements are called and the used storage is deallocated.

:::note
If the elements are pointers, the pointed-to objects are not destroyed.
:::

### Complexity

Linear in the size of the container - **O(size())**.
