# std::forward_list clear() method

<SwitchView content={{
	'since-cpp11': <Method_SinceCpp11 />,
}} />

#### Erases all elements from the container.

:::danger Invalidation
Invalidates any **references**, or **iterators** referring to contained elements.
Any past-the-end iterators remains valid.
:::

### Parameters

**(none)**

### Return value

**(none)**

### Complexity

Linear in the size of the container - **O(size())**.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title="Main.cpp" showLineNumbers
#include <algorithm>
#include <iostream>
#include <forward_list>

int main()
{
    std::forward_list<int> container{1, 2, 3};

    auto print = [](const int& n) { std::cout << " " << n; };

    std::cout << "Before clear:";
    std::for_each(container.begin(), container.end(), print);
    std::cout << '\n';

    std::cout << "Clear\n";
    container.clear();

    std::cout << "After clear:";
    std::for_each(container.begin(), container.end(), print);
    std::cout << '\n';
}
```
