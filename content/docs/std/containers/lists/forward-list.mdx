<details>
<summary>Technical definition of a forward list</summary>
<div>

[`std::forward_list`](#) is a container that supports fast insertion and removal of elements from anywhere in the container.
:::important
Fast random access is not supported.
:::

It is implemented as a singly-linked list. Compared to [`std::list`](forward-list.mdx) this container provides more space efficient storage when bidirectional iteration is not needed.

Adding, removing and moving the elements within the list, or across several lists, does not invalidate the iterators currently referring to other elements in the list. However, an iterator or reference referring to an element is invalidated when the corresponding element is removed (via [`erase_after`](forward-list/erase-after.mdx)) from the list.

[`std::forward_list`](#) meets the requirements of [`Container`](/docs/named_req/Container) (except for the size member function and that [`operator==`](forward-list/operator_eq.mdx)'s complexity is always linear), [`AllocatorAwareContainer`](/docs/named_req/AllocatorAwareContainer) and [`SequenceContainer`](/docs/named_req/SequenceContainer).

</div>
</details>
