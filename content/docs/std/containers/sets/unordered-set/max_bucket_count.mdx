# std::unordered_set max_bucket_count() method

<VersionTabs content={{
    'since-cpp11': <Method_SinceCpp11 />,
    }} />

Returns the maximum number of buckets the container is able to hold due to system or library implementation limitations.

### Parameters

**(none)**

### Return value

Maximum number of buckets.

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp
#include <iostream>
#include <unordered_set>
 
int main()
{
    struct Ha { std::size_t operator()(long x) const { return std::hash<long>{}(x); }; };
 
    auto c1 = std::unordered_set<char, long>{};
    auto c2 = std::unordered_set<long, long>{};
    auto c3 = std::unordered_set<long, long, std::hash<int>>{};
    auto c4 = std::unordered_set<long, long, Ha>{};
 
    std::cout
        << "Max bucket count of\n" << std::hex << std::showbase
        << "c1: " << c1.max_bucket_count() << '\n'
        << "c2: " << c2.max_bucket_count() << '\n'
        << "c3: " << c3.max_bucket_count() << '\n'
        << "c4: " << c4.max_bucket_count() << '\n'
        ;
}```
