# std::unordered_set load_factor() method

<SwitchView content={{
    'since-cpp11': <Method_SinceCpp11 />,

Returns the average number of elements per bucket, that is, [`size()`](/docs/std/containers/sets/unordered-set/size) divided by [`bucket_count()`](/docs/std/containers/sets/unordered-set/bucket_count).

### Parameters

**(none)**

### Return value

Average number of elements per bucket.

### Complexity

Constant - **O(1)**.
