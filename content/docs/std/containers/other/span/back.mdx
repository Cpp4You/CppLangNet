# std::span back() method

<VersionTabs content={{
	'since-cpp20': <Method_SinceCpp20 />,
}} />

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the last character in the span.

:::danger Undefined Behavior
Calling [`back()`](#) on an empty span is <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.
:::

### Parameters

**(none)**

### Return value

Reference to the last element.

### Complexity

Constant - **O(1)**.

### Notes

For a span `s`, the expression `s.back()` is equivalent to `*std::prev(s.end())` and `s[s.size() - 1]`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <span>
#include <iostream>
 
int main() {
    for (std::span str{"ABCDEF"}; !str.empty(); str.remove_suffix(1))
        std::cout << str.back() << ' ' << str << '\n';
}
```

</div>
<div>

```plaintext title="Output"

E ABCDE
D ABCD
C ABC

```
