# std::span begin() method

<SwitchView content={{
	'since-cpp20': <Method_SinceCpp20 />,
}}/>

Returns an <Tooltip title={Term_ContAlias_Iterator}>iterator</Tooltip> to the first element of the view.

If the view is empty, the returned iterator will be equal to [`end()`](end).

### Parameters

**(none)**

### Return value

Iterator to the first element.

### Complexity

Constant - **O(1)**.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <span>
#include <iostream>
 
void print(std::span<const int> sp)
{
    for(auto it = sp.begin(); it != sp.end(); ++it) {
        std::cout << *it << ' ';
    }
    std::cout << '\n';
}
 
void transmogrify(std::span<int> sp)
{
    if (!sp.empty()) {
        std::cout << *sp.begin() << '\n';
        *sp.begin() = 2;
    }
}
 
int main()
{
    int array[] { 1, 3, 4, 5 };
    print(array);
    transmogrify(array);
    print(array);
}
```
