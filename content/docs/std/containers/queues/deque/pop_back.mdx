# std::deque pop_back() method

<SwitchView content={{
	'since-cpp98': <Method_SinceCpp98 />,
}} />

Removes the last element of the container.

Calling [`pop_back`](#) on an empty container results in <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.

<SwitchView content={{
	'since-cpp11': <IteratorDereferenceNote_SinceCpp11 />,
	'until-cpp11': <IteratorDereferenceNote_UntilCpp11 />,
}} />

### Parameters

**(none)**

### Return value

**(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <deque>
#include <iostream>
 
template<typename T>
void print(T const & xs)
{
    std::cout << "[ ";
    for(auto const & x : xs) {
        std::cout << x << ' ';
    }
    std::cout << "]\n";
}
 
int main()
{
    std::deque<int> numbers;
 
    print(numbers); 
 
    numbers.push_back(5);
    numbers.push_back(3);
    numbers.push_back(4);
 
    print(numbers); 
 
    numbers.pop_back();
 
    print(numbers); 
}
```
