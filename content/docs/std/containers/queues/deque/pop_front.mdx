# std::deque pop_back() method

<SwitchView content={{
	'since-cpp98': <Method_SinceCpp98 />,
}} />

Removes the first element of the container.

Calling [`pop_front`](#) on an empty container results in <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.

<SwitchView content={{
	'since-cpp11': <IteratorDereferenceNote_SinceCpp11 />,
	'until-cpp11': <IteratorDereferenceNote_UntilCpp11 />,
}} />

### Parameters

**(none)**

### Return value

**(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <deque>
#include <iostream>
 
int main()
{
    std::deque<char> chars{'A', 'B', 'C', 'D'};
 
    for (; !chars.empty(); chars.pop_front())
    {
        std::cout << "chars.front(): '" << chars.front() << "'\n";
    }
}
```
