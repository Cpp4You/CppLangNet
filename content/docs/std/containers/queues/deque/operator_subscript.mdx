Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the element at specified index `pos`.

:::important
No bounds checking is performed, using an element out of bounds is <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.
:::

### Parameters

- `pos` - position of the element to return

### Return value

Reference to the requested element.

### Exceptions

**(none)**

### Complexity

Constant - **O(1)**.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <deque>
#include <iostream>
 
int main()
{
    std::deque<int> numbers {2, 4, 6, 8};
 
    std::cout << "Second element: " << numbers[1] << '\n';
 
    numbers[0] = 5;
 
    std::cout << "All numbers:";
    for (auto i : numbers) {
        std::cout << ' ' << i;
    }
    std::cout << '\n';
}
```
