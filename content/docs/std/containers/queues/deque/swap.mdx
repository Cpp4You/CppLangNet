# std::deque swap() method

<SwitchView content={{
	'since-cpp17': <Method_SinceCpp17 />,
	'until-cpp17': <Method_UntilCpp17 />
}} />

Exchanges the contents of the container with those of `other`. Does not invoke any move, copy, or swap operations on individual elements.

:::danger Invalidation
The **past-the-end iterator** is invalidated.  
All iterators and references remain valid.
:::

<SwitchView content={{
    'since-cpp11': <PropagateOnContainerChangNote_SineCpp11 />,
}} />

### Parameters

 - `other` - container to exchange the contents with

### Return value

**(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

<SwitchView content={{
    'since-cpp17': <NoexceptSpecification_SinceCpp17 />,
    'until-cpp17': <><b>(none)</b></>

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <deque>

template<class Os, class Co> Os& operator<<(Os& os, const Co& co) {
    os << "{";
    for (auto const& i : co) { os << ' ' << i; }
    return os << " } ";
}

int main()
{
    std::deque<int> a1{1, 2, 3}, a2{4, 5};

    auto it1 = std::next(a1.begin());
    auto it2 = std::next(a2.begin());

    int& ref1 = a1.front();
    int& ref2 = a2.front();

    std::cout << a1 << a2 << *it1 << ' ' << *it2 << ' ' << ref1 << ' ' << ref2 << '\n';
    a1.swap(a2);
    std::cout << a1 << a2 << *it1 << ' ' << *it2 << ' ' << ref1 << ' ' << ref2 << '\n';

    // Note that after swap the iterators and references stay associated with their
    // original elements, e.g. it1 that pointed to an element in 'a1' with value 2
    // still points to the same element, though this element was moved into 'a2'.
}
```
