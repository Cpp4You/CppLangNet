# std::deque clear() method

<VersionTabs content={{
    'since-cpp11': <Method_SinceCpp11 />,
    'until-cpp11': <Method_UntilCpp11 />,
}} />

Erases all elements from the container. After this call, [`size()`](/docs/containers/queues/deque/size) returns zero.

:::danger Invalidation
Invalidates any **references, pointers, or iterators referring to contained elements**.

May also invalidate **past-the-end iterators**.
:::

### Parameters

**(none)**

### Return value

**(none)**

### Complexity

Linear in the size of the container - **O(size())**.

### Exceptions

**(none)**

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <algorithm>
#include <iostream>
#include <deque>

int main()
{
    std::deque<int> container{1, 2, 3};

    auto print = [](const int& n) { std::cout << " " << n; };

    std::cout << "Before clear:";
    std::for_each(container.begin(), container.end(), print);
    std::cout << "\nSize=" << container.size() << '\n';

    std::cout << "Clear\n";
    container.clear();

    std::cout << "After clear:";
    std::for_each(container.begin(), container.end(), print);
    std::cout << "\nSize=" << container.size() << '\n';
}
```
