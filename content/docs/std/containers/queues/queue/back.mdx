# std::queue back()

<SwitchView content={{
		'since-cpp98': <Method_SinceCpp98 />,
	}}/>

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip>  to the last element in the queue.  
This is the most recently pushed element.  

:::note
Effectively calls `c.back()`.
:::

### Parameters

**(none)**

### Return value

Reference to the last element.

### Exceptions

**(none)**

### Complexity

Constant - **(1)**.
