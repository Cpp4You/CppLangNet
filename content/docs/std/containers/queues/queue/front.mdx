# std::queue front

<SwitchView content={{
		'since-cpp98': <Method_SinceCpp98 />,
	}}/>

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip>  to the first element in the queue.  
This element will be the first element to be removed on a call to [`pop()`](/docs/std/containers/queues/queue/pop).  

:::note
Effectively calls `c.front()`.
:::

### Parameters

**(none)**

### Return value

Reference to the first element.

### Exceptions

**(none)**

### Complexity

Constant - **(1)**.
