# std::priority_queue destructor

<VersionTabs content={{
		'since-cpp98': <Method_SinceCpp98 />
	}}/>

Destructs the priority_queue. The destructors of the elements are called and the used storage is deallocated.

:::note
If the elements are pointers, the pointed-to objects are not destroyed.
:::

### Complexity

Linear in the size of the container - **O(size())**.
