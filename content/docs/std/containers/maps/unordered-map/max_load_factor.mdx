# std::unordered_map max_load_factor() method

<SwitchView content={{
    'since-cpp11': <Method_SinceCpp11 />,
    }} />

Manages the maximum load factor (number of elements per bucket).
The container automatically increases the number of buckets if the load factor exceeds this threshold.

 - **(1)** Returns current maximum load factor.
 - **(2)** Sets the maximum load factor to `ml`.

### Parameters

 - `ml` - new maximum load factor setting

### Return value

 - **(1)** Current maximum load factor.
 - **(2)** **(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**
