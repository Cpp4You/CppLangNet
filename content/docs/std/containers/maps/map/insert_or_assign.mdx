# std::map insert_or_assign() method

<SwitchView content={{
    'since-cpp17': <Method_SinceCpp17 />
    }} />

 - **(1-3)**  If a key equivalent to k already exists in the container,
    assigns `std::forward<M>(obj)` to the `mapped_type` corresponding to the key `k`.
    If the key does not exist, inserts the new value as if by [`insert()`](/docs/std/containers/maps/map/insert), constructing it from `value_type(k, std::forward<M>(obj))`.
 - **(2-4)** Same as **(1-3)**, except the mapped value is constructed from `value_type(std::move(k), std::forward<M>(obj))`.

<SwitchView content={{
    'since-cpp20': <> The program is ill-formed if <code>std::is_assignable_v&lt;mapped_type&, M&&&gt;</code> is <code>false</code> </>,
    'until-cpp20': <> The behaviour is undefined if <code>std::is_assignable_v&lt;mapped_type&, M&&&gt;</code> is <code>false</code> </>
    }} />

No iterators or references are invalidated.

### Parameters

 - `key` - the key used both to look up and to insert if not found
 - `hint` - iterator to the position before which the new element will be inserted
 - `obj` - the value to insert or assign

### Return value

 - **(1-2)** The bool component is `true` if the insertion took place and `false` if the assignment took place. The iterator component is pointing at the element that was inserted or updated.
 - **(3-4)** Iterator pointing at the element that was inserted or updated.

### Complexity

 - **(1-2)** Same as for [`emplace()`](/docs/std/containers/maps/map/emplace).
 - **(3-4)** Same as for [`emplace_hint()`](/docs/std/containers/maps/map/emplace_hint).

### Exceptions

**(none)**

### Notes 
[`insert_or_assign()`](#) returns more information than [`operator[]`](/docs/std/containers/maps/map/operator_subscript) and does not require default-constructibility of the mapped type.

Feature testing macro: [`__cpp_lib_map_try_emplace`](#).

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <map>
#include <string>
 
auto print_node = [](const auto &node) {
    std::cout << "[" << node.first << "] = " << node.second << '\n';
};
 
auto print_result = [](auto const &pair) {
    std::cout << (pair.second ? "inserted: " : "assigned: ");
    print_node(*pair.first);
};
 
int main()
{
    std::map<std::string, std::string> myMap;
 
    print_result( myMap.insert_or_assign("a", "apple"     ) );
    print_result( myMap.insert_or_assign("b", "banana"    ) );
    print_result( myMap.insert_or_assign("c", "cherry"    ) );
    print_result( myMap.insert_or_assign("c", "clementine") );
 
    for (const auto &node : myMap) { print_node(node); }
}
```
