# std::map begi() method

<SwitchView content={{
		'since-cpp11': <Method_SinceCpp11 />,
		'until-cpp11': <Method_UntilCpp11 />,
	}}/>

Returns an <Tooltip title={Term_ContAlias_Iterator}>iterator</Tooltip> to the first element of the map.
If the array is empty, the returned iterator will be equal to [`end()`](/docs/std/containers/maps/map/end).

### Parameters

**(none)**

### Return value

Iterator to the first element.

### Exceptions

**(none)**

### Complexity

Constant.

### Difference between begin and cbegin

For a const container `c`, begin and cbegin are the same - `c.begin() == c.cbegin()`

For non-const container of type `c` they return different iterators:

<Tabs>

<TabItem value="begin" label="begin" default>

```cpp
#include <map>

int main()
{
    std::map<int, float> map = { {1, 1.f}, {2, 3.f}, {5, 8.f} };
    auto it = map.begin(); // Type: std::map<int, float>::iterator
    // highlight-next-line
    *it = 5; // ✔ Ok
}
```

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <map>
 
int main() {
  std::map<int, float> num_map;
  num_map[4] = 4.13;
  num_map[9] = 9.24;
  num_map[1] = 1.09;
  // calls a_map.begin() and a_map.end()
  for (auto it = num_map.begin(); it != num_map.end(); ++it) {
    std::cout << it->first << ", " << it->second << '\n';
  }
}
```

</div>
