### Technical details

<details>
<summary>Technical definition of a string view</summary>
<div>

The class template `basic_string_view` describes an object that can refer to a constant contiguous sequence of char-like objects with the first element of the sequence at position zero.

A typical implementation holds only two members: a pointer to constant `CharT` and a size.

<Since v='cpp11'> Every specialization of <a href="string"><code>std::basic_string_view</code></a> is a <a href="/docs/named_req/TriviallyCopyable"><code>TriviallyCopyable</code></a> type. </Since>

</div>
</details>

<details>
<summary>Named requirements</summary>
<div>

`std::basic_string_view` meets the requirements of:
- <Since v="cpp23"><a href="/docs/named_req/TriviallyCopyable">TriviallyCopyable</a></Since>

	-
		:::note
		Specializations of `std::basic_string_view` are already trivially copyable types in all existing implementations, even before the formal requirement introduced in C++23.
		:::

</div>
</details>
