Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the element at specified index `pos`.

:::caution
No bounds checking is performed.
:::

<SwitchView content={{
	'since-cpp11': <Explanation_SinceCpp11 />,
	'until-cpp11': <Explanation_UntilCpp11 />,
}} />

### Parameters

- `pos` - position of the character to return

### Return value

Reference to the requested character.

### Exceptions

**(none)**

### Complexity

Constant - **O(1)**.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <string>
 
int main()
{
    std::string const e("Exemplar");
    for (unsigned i = e.length() - 1; i != 0; i /= 2)
        std::cout << e[i];
    std::cout << '\n';
 
    const char* c = &e[0];
    std::cout << c << '\n'; // print as a C string
 
    //Change the last character of s into a 'y'
    std::string s("Exemplar ");
    s[s.size()-1] = 'y'; // equivalent to s.back() = 'y';
    std::cout << s << '\n';
}
```
