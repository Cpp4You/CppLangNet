# std::string swap() method

<SwitchView content={{
	'since-cpp20': <Method_SinceCpp20 />,
	'since-cpp17': <Method_SinceCpp17 />,
	'until-cpp17': <Method_UntilCpp17 />
}} />

Exchanges the contents of the string with those of `other`.

:::caution Possible invalidation
**All iterators** and **references** may be invalidated.
:::

<SwitchView content={{
	'since-cpp11': <PropagateOnContainerSwap_Note_SinceCpp11 />
}} />

### Parameters

 - `other` - the string to exchange the contents with

### Return value

**(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

<SwitchView content={{
	'since-cpp17': <NoexceptSpecification_SinceCpp17 />
}} />

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <string>
#include <iostream>
 
int main() 
{
    std::string a = "AAA";
    std::string b = "BBB";
 
    std::cout << "before swap" << '\n';
    std::cout << "a: " << a << '\n';
    std::cout << "b: " << b << '\n';
 
    a.swap(b);
 
    std::cout << "after swap" << '\n';
    std::cout << "a: " << a << '\n';
    std::cout << "b: " << b << '\n';
}
```
