# std::string back() method

<SwitchView content={{
	'since-cpp20': <Method_SinceCpp20 />,
	'since-cpp11': <Method_SinceCpp11 />,
}} />

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the last character in the string.

:::danger Undefined Behavior
Calling [`back()`](#) on an empty string is <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.
:::

### Parameters

**(none)**

### Return value

Reference to the last character.

### Complexity

Constant - **O(1)**.

### Notes

For a string `str`, the expression `str.back()` is equivalent to `*std::prev(str.begin())` and `str[str.size() - 1]`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <string>
 
int main()
{
  {
    std::string s("Exemplary");
    char& back = s.back();
    back = 's';
    std::cout << s << '\n'; // "Exemplars"
  }
 
  {
    std::string const c("Exemplary");
    char const& back = c.back();
    std::cout << back << '\n'; // 'y'
  }
}
```

</div>
<div>

```plaintext title="Output"

y
```
