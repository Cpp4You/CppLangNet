# std::string copy() method

<SwitchView content={{
	'since-cpp20': <Method_SinceCpp20 />,
	'until-cpp20': <Method_UntilCpp20 />,
}}/>

Copies a substring **[ pos, pos + count )** to character string pointed to by `dest`.  
If the requested substring lasts past the end of the string, or if `count == npos`, the copied substring is **[ pos, size() )**.


The resulting character string **is not** null-terminated.

### Parameters

 - `dest` - pointer to the destination character string
 - `count` - length of the substring
 - `pos` - position of the first character to include

### Return value

Number of characters copied.

### Complexity

Linear in `count` - **O(count)**.

### Exceptions

Throws [`std::out_of_range`](#) if `pos > size()`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp
#include <string>
#include <iostream>
 
int main()
{
  std::string foo("quuuux");
  char bar[7]{};
  foo.copy(bar, sizeof bar);
  std::cout << bar << '\n';
}
```

</div>
