# std::string pop_back() method

<SwitchView content={{
	'since-cpp11': <Method_SinceCpp11 />,
	'since-cpp20': <Method_SinceCpp20 />,
}} />

Removes the last character from the string as if by
```cpp
erase(end()-1)
````

:::danger Undefined Behavior
The <Tooltip title={Term_UndefinedBehaviour}>behavior is undefined</Tooltip> if the string is empty.
:::

### Parameters

**(none)**

### Return value

**(none)**

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**

### Example

<div>

```cpp title='Main.cpp' showLineNumbers
#include <cassert>
#include <string>
#include <iomanip>
#include <iostream>
 
int main()
{
    std::string str("Short string!");
    std::cout << "before=" << quoted(str) << '\n';
    assert(str.size() == 13);
 
    str.pop_back();
    std::cout << " after=" << quoted(str) << '\n';
    assert(str.size() == 12);
 
    str.clear();
//  str.pop_back(); // UB!
}
```
