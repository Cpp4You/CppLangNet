# std::string front() method

<SwitchView content={{
	'since-cpp20': <Method_SinceCpp20 />,
	'since-cpp11': <Method_SinceCpp11 />,
}} />

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the first character in the string.

:::danger Undefined Behavior
Calling [`front`](#) on an empty string is <Tooltip title={Term_UndefinedBehaviour}>undefined</Tooltip>.
:::

### Parameters

**(none)**

### Return value

Reference to the first character.

### Complexity

Constant - **O(1)**.

### Notes

For a string `str`, the expression `str.front()` is equivalent to `*str.begin()` and `str[0]`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <string>
 
int main()
{
  {
    std::string s("Exemplary");
    char& f = s.front();
    f = 'e';
    std::cout << s << '\n'; // "exemplary"
  }
 
  {
    std::string const c("Exemplary");
    char const& f = c.front();
    std::cout << &f << '\n'; // "Exemplary"
  }
}
```

</div>
