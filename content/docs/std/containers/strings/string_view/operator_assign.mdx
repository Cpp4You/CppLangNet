# std::string_view operator=

<SwitchView content={{
    'since-cpp17': <Method_SinceCpp17 />,
}} />

Replaces the view with that of `view`.

### Parameters

 - `view` - view to copy

### Return value

`*this`

### Complexity

Constant - **O(1)**.

### Exceptions

**(none)**

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <iostream>
#include <string_view>
int main()
{
    std::string_view v = "Hello, world";
    v = v.substr(7);
    std::cout << v << '\n';
}
```

</div>
