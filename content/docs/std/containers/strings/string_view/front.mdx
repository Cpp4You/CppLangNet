# std::string_view front() method

<SwitchView content={{
	'since-cpp17': <Method_SinceCpp17 />,
}} />

Returns a <Tooltip title={Term_ContAlias_Reference}>reference</Tooltip> to the first character in the view.

:::danger Undefined Behavior
Calling [`front()`](#) on an empty view is <Tooltip title={Term_UndefinedBehaviour}>undefined behavior</Tooltip>.
:::

### Parameters

**(none)**

### Return value

Reference to the first character.

### Complexity

Constant - **O(1)**.

### Notes

For a view `str`, the expression `str.front()` is equivalent to `*str.begin()` and `str[0]`.

### Example

<Columns columns={['8fr', '4fr']}>
<div>

```cpp title='Main.cpp' showLineNumbers
#include <string_view>
#include <iostream>
 
int main() {
    for (std::string_view str{"ABCDEF"}; !str.empty(); str.remove_prefix(1))
        std::cout << str.front() << ' ' << str << '\n';
}
```

</div>
