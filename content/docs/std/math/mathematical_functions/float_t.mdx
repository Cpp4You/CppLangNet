# std::float_t; std::double_t

### Description

The types <span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span> are floating types at least as wide as <span class='col-lb'>float</span> and <span class='col-lb'>double</span>, 
respectively, and such that <span class='col-lb'>double_t</span> is at least as wide as <span class='col-lb'>float_t</span>. 

The value of `FLT_EVAL_METHOD` determines the types of <span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span>.

`FLT_EVAL_METHOD`	Explanation: <br/>
`0`	<span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span> are equivalent to <span class='col-lb'>float</span> and <span class='col-lb'>double</span>, respectively<br/>
`1`	both <span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span> are equivalent to <span class='col-lb'>double</span><br/>
`2`	both <span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span> are equivalent to <span class='col-lb'>long double</span><br/>
other	both <span class='col-lb'>float_t</span> and <span class='col-lb'>double_t</span> are implementation defined<br/>

### Declarations 
```cpp title="Since C99"
typedef /*implementation defined*/ float_t

typedef /*implementation defined*/ double_t
```

### Examples

<Columns spacing = "20px" columns={['6fr', '4fr']}>
<div>

```cpp
#include <float.h>
#include <math.h>
#include <stdio.h>
 
int main(void)
{
    printf("%d\n", FLT_EVAL_METHOD);
    printf("%zu  %zu\n", sizeof(float),sizeof(float_t));
    printf("%zu  %zu\n", sizeof(double),sizeof(double_t));
    return 0;
}

```
