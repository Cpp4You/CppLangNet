# std::auto_ptr&lt;T>::release

`Deprecated in C++11` `Removed in C++17`
```cpp
T* release() throw();
```
		
Releases the held pointer. After the call `*this` holds the null pointer.

### Parameters
		
(none)

### Return value

`get()`.
