# <small>C++ named requirements:</small> *DefaultConstructible*
Specifies that an instance of the type can be default constructed.

### Requirements
The type T satisfies *DefaultConstructible* if <br/>
Given
* `u`, an arbitrary identifier

The following expressions must be valid and have their specified effects

### Notes

For objects of non-aggregate class type, a public [default constructor](/docs/std/language/default_constructor) must be defined 
(either user-defined or implicitly defined) to satisfy *DefaultConstructible*.

Non-const objects of [non-class object type](/docs/std/language/type) are always *DefaultConstructible*.

Const non-class types are not *DefaultConstructible*.

Const aggregate types are not *DefaultConstructible* if any of their members is an object of non-class type.

Non-object types (function types, reference types, and the (possibly cv-qualified) type void) 
as well as the const non-object types are never *DefaultConstructible*
