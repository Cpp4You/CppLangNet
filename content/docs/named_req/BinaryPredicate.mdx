# <small>C++ named requirements:</small> *BinaryPredicate*
***BinaryPredicate*** is a set of requirements expected by some of the standard library facilities from the user-provided arguments.

Given a *BinaryPredicate* `bin_pred` and a pair of iterators `iter1` and `iter2` or an iterator `iter` and a <Since v='cpp20'>(possibly const)</Since> value `value`, 
the expression `bin_pred(*iter1, *iter2)` or, respectively, `bin_pred(*iter, value)`, must be [contextually convertible](/docs/std/language/implicit_conversion) to `bool`.

In addition, evaluation of that expression is not allowed to call non-const member functions of the dereferenced iterators; <Since v='cpp20'> syntactically, 
the predicate must accept const object arguments, with the same behavior regardless of whether its arguments are const or non-const</Since>.

### Requirements
* [Predicate](/docs/named_req/Predicate)
* [CopyConstructible](/docs/named_req/CopyConstructible) (unless otherwise specified)

### Standard library
The following standard library facilities expect a *BinaryPredicate* which isn't a [Compare](/docs/named_req/Compare) type.
