# <small>C++ named requirements:</small> *Callable*

A ***Callable*** type is a type for which the [INVOKE](/docs/std/utility/functional) and [INVOKE&lt;R&gt;](/docs/std/utility/functional) operations 
(used by, e.g., [std::function](/docs/std/utility/functional/function), [std::bind](/docs/std/utility/functional/bind), 
and [std::thread&#58;:thread](/docs/std/thread/_thread/thread)) are applicable.

<Since v='cpp17'>INVOKE can be performed explicitly using the library function <a href="/docs/std/utility/functional/invoke">std::invoke</a>.</Since> <br/>
<Since v='cpp23'>INVOKE&lt;R&gt; can be performed explicitly using the library function <a href="/docs/std/utility/functional/invoke">std::invoke_r</a>.</Since>

### Requirements
The type T satisfies *Callable* if <br/>
Given

* `f`, an object of type `T`
* `ArgTypes`, suitable list of argument types
* `R`, suitable return type

The following expressions must be valid:

### Notes
[Pointers to data](/docs/std/language/pointer#Pointers_to_data_members) members are *Callable*, even though no function calls take place.
