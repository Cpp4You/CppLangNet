# <Since v='cpp17'><small>C++ named requirements:</small> *ContiguousContainer*</Since>
A ***ContiguousContainer*** is a [*Container*](/docs/named_req/Container) that stores objects in contiguous memory locations.

### Requirements
The type X satisfies *ContiguousContainer* if

* The type X satisfies [*Container*](/docs/named_req/Container)
* The type X supports [*LegacyRandomAccessIterators*](/docs/named_req/LegacyRandomAccessIterators)
* The member types `X::iterator` and `X::const_iterator` 
are <Until v='cpp20'><a href="/docs/named_req/LegacyContiguousIterators"><i>LegacyContiguousIterators</i></a></Until> <Since v='cpp20'><a href="/docs/std/iterator/contiguous_iterator"><i>contiguous_iterators</i></a></Since>
