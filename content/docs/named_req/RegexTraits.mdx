# <Since v='cpp11'><small>C++ named requirements:</small>*RegexTraits*</Since>

***RegexTraits*** is a traits class that abstracts operations needed for a given character type when as template type parameters 
for <code><a href="/docs/std/regex/basic_regex">std::basic_regex</a></code>.

### Requirements
Given

* `CharT`, a character type
* `X`, a *RegexTraits* type for type **CharT**
* `u`, an object of type **X**
* `v`, an object of type <code><a>const</a> <b>X</b></code>
* `p`, a value of type <code><a>const</a> <b>CharT\*</b></code>
* `I1`, `I2`, input iterators
* `F1`, `F2`, forward iterators
* `c`, a value of type <code><a>const</a> <b>CharT</b></code>
* `s`, an object of type X::string_type
* `cs`, an object of type <code><a>const</a> X::string_type</code>
* `b`, a value of type <a>bool</a>
* `I`, a value of type <a>int</a>
* `cl`, an object of type `X::char_class_type`
* `loc`, an object of type `X::locale_type`
