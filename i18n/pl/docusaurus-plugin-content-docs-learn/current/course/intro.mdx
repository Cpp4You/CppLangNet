## Pierwszy program

Jeśli masz już gotowy swój edytor kodu, możesz przystąpić do napisania pierwszego programu.

Przyjęło się, że podczas nauki nowego języka programowania, pierwszą rzeczą, którą się robi
to prosta aplikacja wyświetlająca tekst **"Witaj, świecie!"** ("Hello, World!" po angielsku).

Po stworzeniu projektu, w Twoim środowisku, najpewniej został automatycznie dodany plik o rozszerzeniu `.cpp`.
Najczęściej nazywa się on `main.cpp` (czasami `Source.cpp` lub `NazwaProjektu.cpp`).
Jeśli takiego nie ma, to stwórz go i nazwij właśnie `main.cpp` (*main* - z ang.: główny).

:::tip Nowe pliki
Jeśli nie wiesz jak dodać plik do projektu, zajrzyj w dział <a href="/tools" target="_blank">narzędzia</a>
i znajdź poradnik do wybranego przez siebie edytora.
:::

To właśnie w nim będziemy zapisywali póki co cały kod.

### Kod aplikacji

Wklej następujący kod i go uruchom:

<Columns spacing={40} columns={['8fr', '4fr']}>
<div>

```cpp title="main.cpp"
#include <iostream>

int main()
{
	std::cout << "Witaj, świecie!";
}
```

</div>

<div>

```plaintext title="Wynik (konsola)"
Witaj, świecie!
```

</div>
</Columns>


Powinieneś zaobserwować efekt taki jak powyżej.

:::important Polskie znaki
Jeśli używasz systemu Windows i korzystasz ze środowiska zainstalowanego na własnym komputerze
to najprawdopodobniej zamiast polskich znaków pojawią się różne "krzaczki".
Jest to znany problem, jednak nie przejmuj się nim na tym etapie nauki.
Zamiast tego napisz **"Witaj, swiecie!"** - bez polskich znaków.
:::

Omówienie podanego wyżej kodu znajdziesz w następnej lekcji 🙂
