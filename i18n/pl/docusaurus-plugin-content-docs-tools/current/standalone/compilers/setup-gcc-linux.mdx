# Instalacja GCC na Linuksie

Jeśli masz Linuxa opartego na dystrybucji Ubuntu otwórz terminal i wpisz

```bash
> sudo apt update
> sudo apt install g++
```

Następnie zweryfikuj czy komenda `g++ --version` wypisuje Ci wersję kompilatora. Przykładowy poprawny output:

```bash
> g++ --version
g++ (GCC) 10.3.0
Copyright (C) 2020 Free Software Foundation, Inc.
This is free software; see the source for copying conditions.  There is NO
warranty; not even for MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.
```

W zależności od wersji kompilatora różne nowości z kolejnych standardów języka mogą być niedostępne. Zapoznaj się z [tą tabelą na cppreference.org](https://en.cppreference.com/w/cpp/compiler_support)
