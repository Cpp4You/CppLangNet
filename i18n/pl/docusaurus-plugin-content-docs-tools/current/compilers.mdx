# Wybór kompilatora

**Kompilator** to narzędzie niezbędne do programowania w C++.

## Czym jest kompilator

Kompilator tłumaczy kod napisany w języku C++, na format danych który rozumieją komputery.
Kod C++ nie jest bezpośrednio zrozumiały dla komputerów, dlatego musi zostać
skonwertowany.
