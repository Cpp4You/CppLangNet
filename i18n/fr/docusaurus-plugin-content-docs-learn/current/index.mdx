Vous êtes dans la section de l'apprentissage de programmation. Vous pouvez trouver ici un cour sur le langage de C++, avec d'autres pages, dans lesquelles le sujet fait le tour de la programmation dans le langage en question.

Avant de passer au cours, renseignez-vous sur quoi est la programmation en général.

## C'est quoi la programmation

La programmation est utilisée dans l'automatisation des tâches grâce aux machines, qui effectuent des instructions. 
Dans le cours C++, vous allez apprendre comment créer un programme, qui peut être utilisé par un ordinateur.
Grâce à ce cours vous pourrez créer une application, qui par exemple:

- fais des calculs à ta place ➕ ➖ ✖ ➗
- est un jeu vidéo 🎮 😄
- te remplacera dans des routines, monotones dans le domaine informatique 🖨 ☕

### Cela ressemble à quoi

**L'informaticien écrit le code.** Nous allons te montrer comment coder de manière, à ce que cela soit cohérent avec les règles du langage de C++ et nous allons t'apprendre les bonnes pratiques de programmation.

Voici un bloc de code facile écrit dans le langage C++, qui calcule l'aire du trapèze:

<CustomCodeBlock maxLines={15} resizable>

```cpp title="main.cpp" showLineNumbers
#include <iostream>

int main()
{
	float a = 15;
	float b = 10;
	float h = 5;

	std::cout << "LE LOGICIEL CALCULE L'AIRE DU TRAPÈZE\n";

	std::cout << "Donnez la valeur de a: ";
	std::cin >> a;

	std::cout << "Donnez la valeur de b: ";
	std::cin >> b;

	std::cout << "Donnez la valeur de h: ";
	std::cin >> h;

	if (a > 0 && b > 0 && h > 0)
	{
		std::cout << "L'aire du trapèze avec les valeurs:\n";
		std::cout << "a = " << a << "\n";
		std::cout << "b = " << b << "\n";
		std::cout << "h = " << h << "\n";
		std::cout << "est égale: ";
		std::cout << (a + b) * h / 2;
	}
	else
	{
		std::cout << "Toutes les valeurs doivent être positives!\n";
	}
}
```

</CustomCodeBlock>

Si vous voulez voir comment ça marche, **la réponse est dans le cours**!

Le code C++ n'est pas directement lu par un ordinateur. Avant la possibilité de lecture, le code doit passer par dit un **compilateur** (nous en parlant par la suite), qui vérifie d'abord si code respecte les règles du C++, et si cela est le cas, le traduit dans un langage compris par les ordinateurs.

### À quoi sert le C++

Le C++ est un langage de programmation **d'utilisation générale**. Nous pouvons l'utiliser pour créer, vraiment tout ce qui nous passe par la tête. Par contre vaut bien savoir où l'utiliser au mieux:

- dans le développement des jeux 🕹 (par ex. à l'aide du moteur Unreal Engine 4)
- création d'application de bureau 💻 (dans ceux, où l'optimisation joue un rôle important)
- dans la création des serveurs ⏫ 🖥 ⏬
- dans la création des microcontrôleurs 🤖

Actuellement C++ n'est pas la meilleure des solutions si vous voulez créer:

- un site web 🌐 (utilisez le JavaScript à la place)
- Applications mobiles 📱 et de bureau 🏢, qui demandent pas beaucoup d'optimisation

### Les choses à s'en rappeler

L'apprentissage de programmation est un processuce long, et le C++ est **très** vaste et étendu. Cela ne veut pas dire, que l'apprentissage doit être dur -
au contraire! Ce site a été créer dans le but principal, casser le sérotype du C++, étant la peur des étudiants.

Étendu ne veut pas dire difficile!
